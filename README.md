## 🚀 Featured Projects

### 1. [Pixel Pulse](https://play.google.com/store/apps/details?id=io.github.fertwbr.pixelpulse)
- **Status:** Actively in Development 
- **Description:** A smart sound meter and hearing health coach for Android. It features an expressive real-time gauge that adapts to noise levels, detailed session history with interactive charts, and background monitoring to ensure hearing safety.
- **Tech Stack:** **Kotlin, Jetpack Compose, Material 3, Hilt, WorkManager (Background Monitoring), AudioRecord & JTransforms (FFT), Room Database, Google Play Billing.**

### 2. [Pixel Compass](https://play.google.com/store/apps/details?id=io.github.fertwbr.pixelcompass)
- **Status:** Actively in Development 
- **Description:** A premium navigation companion for **Android and Wear OS**. It features a highly interactive UI, configurable home screen widgets (Glance), environmental data cards (Weather, Altitude, UV), and a precise level tool. Built with a clean, multi-module architecture.
- **Tech Stack:** **Kotlin, Jetpack Compose, Wear OS, Jetpack Glance (Widgets), Retrofit & Moshi, SensorManager, Fused Location, Hilt.**

### 3. [boxIdea](https://github.com/fertwbr/boxIdea)
- **Status:** Completed
- **Description:** A collaborative system where users can submit work-related experiences and ideas, categorized by themes, to foster innovation and knowledge sharing within teams.
- **Tech Stack:** **Java, Spring Boot, PostgreSQL, JWT Authentication**

### 4. [LocateCEP](https://github.com/fertwbr/LocateCEP)
- **Status:** Completed
- **Description:** A Java-based tool for locating UF/City via ZIP code.
- **Tech Stack:** Java

### 5. [OpenLibrary](https://github.com/fertwbr/OpenLibrary)
- **Status:** Completed

- **Tech Stack:** **Java, Spring Boot, PostgreSQL**
